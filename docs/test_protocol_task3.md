# Протокол тестирования — Задание 3

## 1. Общие сведения

**Объект тестирования:** модифицированный модуль учета заявок (роль менеджера по качеству).  
**Средство тестирования:** `pytest` + встроенный тестовый клиент Flask.  
**Дата:** 2025‑12‑16.  
**Ответственный:** студент (исполнитель).

## 2. Набор тестов

| № | Наименование | Входные данные | Ожидаемый результат |
|---:|---|---|---|
| 1 | Запрос помощи специалиста | сообщение запроса помощи | запрос появляется у менеджера (панель/уведомления) |
| 2 | Продление срока без согласования | `due_date` без `customer_agreed` | операция запрещена, выводится сообщение об ошибке |
| 3 | Продление срока с согласованием | `due_date` + `customer_agreed` | срок сохраняется, пишется история сроков |
| 4 | QR‑код | запрос `/tickets/<id>/qr` | возвращается SVG `image/svg+xml` |
