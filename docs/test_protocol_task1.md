# Протокол тестирования (Задание 1)

**Проект:** модуль учета заявок на ремонт климатического оборудования  
**Инструмент тестирования:** pytest  
**Окружение:** Python 3.12, Flask 3.1, SQLite  
**Команда запуска:** `pytest -q`  

## Таблица тест-кейсов

| ID | Проверяемая функция | Входные данные | Шаги | Ожидаемый результат | Фактический результат | Статус |
|---|---|---|---|---|---|---|
| TC‑01 | `calculate_statistics()` | 2 завершенные заявки + 1 незавершенная | Запустить `pytest -q` | Верные: количество=2, среднее время, распределение по категориям | Соответствует ожидаемому | PASS |
| TC‑02 | Сквозной сценарий заявок | Данные заявки + комментарий + комплектующая + завершение | Запустить `pytest -q` | Создается заявка, добавляется комментарий/комплектующая, статус меняется на «Завершена», оператор получает уведомление | Соответствует ожидаемому | PASS |

## Примечания
- При ручном тестировании через браузер рекомендуется дополнительно проверить: поиск по параметрам, подтверждение удаления, сообщения об ошибках (пустые поля/некорректный телефон), разграничение прав ролей.
